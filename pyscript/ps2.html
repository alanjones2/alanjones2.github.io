Weather Graphs

Some graphs about the weather in London in 2018

Maximum temperature each month
Minimum temperature each month
Centimetres of rain each month
Hours of sun each month
# Import libraries import pandas as pd import matplotlib.pyplot as plt ## Get the data from pyodide.http import open_url url = 'https://raw.githubusercontent.com/alanjones2/ajbooks/main/data/londonweather2018.csv' url_content = open_url(url) weather = pd.read_csv(url_content) fig, ax = plt.subplots() weather.plot(y='Tmax', figsize=(5,2),ax=ax) fig fig, ax = plt.subplots() weather.plot(y='Tmin', figsize=(4,2), ax=ax) fig fig, ax = plt.subplots() weather.plot.bar(y='Rain', figsize=(4,2), ax=ax) fig fig, ax = plt.subplots() weather.plot.bar(y='Sun', figsize=(4,2), ax=ax) fig
